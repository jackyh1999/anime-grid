# 🤖「動畫生涯個人喜好表生成器」

一個生成 動畫生涯個人喜好表 的網頁工具

forked from https://github.com/itorr/anime-grid

輸出繁體中文圖片，還沒買網域僅能下載使用

原作者GitHub https://github.com/itorr/anime-grid

- 最佳畫面

- 最感動

- 最虐心
- 最被低估

- 最離譜
- 最討厭
